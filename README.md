🚖 Taxi Data Analysis Dashboard (Power BI)
This project presents an interactive **Power BI dashboard** built using a taxi trip dataset, focusing on analyzing key ride metrics such as fare amount, passenger count, and geospatial data (pickup & drop-off locations).

📊 Project Overview
The dashboard provides a comprehensive view of taxi trip patterns over time, helping uncover trends in revenue, ride distribution, and passenger behavior.

🔍 Key Insights
* **Total Fare Amount:** 2.27M+ analyzed across trips
* **Passenger Trends:** Visualization of passenger count variations over time
* **Location Analysis:** Aggregated pickup and drop-off latitude & longitude insights
* **Time-Based Analysis:** Ride activity segmented by pickup datetime
* **Comparative Metrics:** Multiple measures (fare, passengers, coordinates) combined for deeper insights

📈 Dashboard Features
* Bar charts showing pickup longitude trends over time
* Line/area chart comparing passenger count and location metrics
* Donut chart summarizing distribution of key variables
* KPI card highlighting total revenue (fare amount)
* Filter panel for dynamic date/time-based analysis
* Gauge visualization for quick metric overview

🛠️ Tools & Technologies
* **Power BI** – Data visualization and dashboard creation
* **Excel / CSV Dataset** – Data source
* **Data Modeling** – Aggregations and calculated measures

🎯 Objective
The goal of this project is to:
* Practice **data visualization and storytelling**
* Extract meaningful insights from raw transportation data
* Build an interactive dashboard for decision-making support

📌 Use Cases
* Ride-sharing business analysis
* Demand and revenue trend identification
* Location-based service optimization
